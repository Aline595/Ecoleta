# Ecoleta
## O que é?
O Ecoleta é uma aplicação Web para ajudar pessoas a encontrarem pontos de coleta para reciclagem.
Essa aplicação foi construída durante a semana Next Level Week distribuída pela Rocketseat.

## Tecnologia Usadas
- TypeScript
- Node.js
- ReactJS
- React Native
